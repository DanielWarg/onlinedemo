# Starta Fort Knox Local - Snabbguide

## Rätt Path

Fort Knox Local finns i:
```
/Users/evil/Desktop/EVIL/PROJEKT/Arbetsytan/fortknox-local
```

ELLER relativt från hemkatalogen:
```
~/Desktop/EVIL/PROJEKT/Arbetsytan/fortknox-local
```

## Starta Tjänsterna

### Terminal 1 - llama.cpp server:
```bash
cd ~/Desktop/EVIL/PROJEKT/Arbetsytan/fortknox-local
./start_llama_server.sh
```

### Terminal 2 - Fort Knox Local:
```bash
cd ~/Desktop/EVIL/PROJEKT/Arbetsytan/fortknox-local
./start.sh
```

## Snabbkommando (kopiera och klistra in)

```bash
cd ~/Desktop/EVIL/PROJEKT/Arbetsytan/fortknox-local && ./start_llama_server.sh
```
